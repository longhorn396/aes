# CS 361 AES Assignment

## How to Run

### Requirements

* Python 3 (in PATH as python)
* make

### Make rules

* test_128 - encodes and decodes the message in test_in
* test_256 - encodes and decodes the message in big_in

### Manually

Run `python aes.py --help` for complete list of options
